# Nano Banana Image Prompt Library

This repository curates high-fidelity image prompts for Nano Banana and Nano Banana Pro models.  
Prompts are sourced from community discussions across X (Twitter), WeChat, Replicate forums, independent prompt engineers, and other public repositories.

Whether you’re looking for photorealistic portraits, stylized aesthetics, or complex creative experiments, you will find a broad set of effective examples here to unlock the expressive potential of generative models.

---

## 📌 What’s Inside

This collection includes prompts organized by:

- **Photorealistic Portraits**  
- **Creative & Stylized Outputs**  
- **YouTube Thumbnails & Graphic Layouts**  
- **Product & Commercial Photography**  
- **Image-to-Image Refinement Patterns**

Each prompt includes a concise description and structured example for easy reuse.

---

## 🛠 How to Use

1. Browse the categories in the `prompts/` directory.
2. Copy the prompt text relevant to your use case.
3. Paste into your Nano Banana workflow (2K/4K) or image-to-image interface.
4. Adjust resolution, composition, and style modifiers as needed.

## 🧠 Notes & Best Practices

✔ Most stable prompts follow a structured pattern with:  
*Subject → Style anchor → Lighting → Camera hint → Detail modifiers → Composition context*

✔ For higher resolutions (4K), add micro-detail descriptors and explicit realism anchors.

✔ When refining existing images, use image-to-image suffixes that emphasize texture clarity and lighting correction.
