# Nano Banana Image Prompt Library

A collection of prompt engineering experiments for AI image generation using Google Nano Banana.

This repository documents practical prompt patterns, resolution strategies, and style tuning approaches discovered while building AI image workflows.

---

## What’s Inside

- Text-to-image prompt patterns
- Image-to-image enhancement prompts
- 2K vs 4K resolution strategies
- Structured prompt templates
- Style anchoring techniques

---

## Why This Exists

Working with AI image models requires more than just descriptive text.

Small wording changes can significantly impact:

- Lighting realism
- Detail density
- Composition balance
- Output sharpness

This repository collects tested patterns and experiments to improve consistency and visual quality.
